# Bailey's Background Remover

A simple web application that removes backgrounds from images using machine learning. Built with Python (FastAPI) for the backend and vanilla HTML/CSS/JavaScript for the frontend.

- Let the user select an image
- Sent the image to the backend
- Recieve the processed image
- Show it on the screen
- Allow the user to download it
